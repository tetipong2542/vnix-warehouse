   คู่มือการใช้งานบน Windows

📋 ขั้นตอนการใช้งาน (ครั้งแรก)
========================================

1. ติดตั้ง Python (ถ้ายังไม่มี)
   - ดาวน์โหลดจาก: https://www.python.org/downloads/
   - เลือก Python 3.8 หรือใหม่กว่า
   - ⚠️ สำคัญ: ติ๊กถูก "Add Python to PATH" ตอนติดตั้ง

2. ดับเบิลคลิก "setup.bat"
   - ระบบจะติดตั้ง dependencies อัตโนมัติ

3. เปิด Firewall (เลือก 1 วิธี)

   วิธีที่ 1: ใช้ Script (แนะนำ)
   - คลิกขวาที่ "open_firewall.bat"
   - เลือก "Run as administrator"

   วิธีที่ 2: เปิดด้วยมือ
   - กด Win + R
   - พิมพ์ "firewall.cpl" แล้วกด Enter
   - คลิก "Advanced settings"
   - คลิก "Inbound Rules" → "New Rule"
   - เลือก "Port" → กรอก 8000 → Allow

4. ดับเบิลคลิก "start.bat"
   - ระบบจะแสดง IP Address ให้อัตโนมัติ
   - เช่น: http://192.168.1.100:8000

5. เปิดบราวเซอร์จากเครื่องอื่นในเครือข่าย
   - พิมพ์ IP Address ที่แสดงใน start.bat


========================================
📱 การใช้งานปกติ (หลังติดตั้งแล้ว)
========================================

✅ รันระบบ:
   - ดับเบิลคลิก "start.bat"

❌ หยุดระบบ:
   - กด Ctrl+C ในหน้าต่าง Command Prompt
   - หรือปิดหน้าต่างได้เลย


========================================
🔧 แก้ไขปัญหา
========================================

❓ ปัญหา: "python is not recognized"
✅ แก้ไข: ติดตั้ง Python ใหม่และติ๊กถูก "Add Python to PATH"

❓ ปัญหา: เครื่องอื่นเข้าไม่ได้
✅ แก้ไข:
   1. ตรวจสอบว่าเครื่องทั้งสองอยู่ใน WiFi เดียวกัน
   2. รัน "open_firewall.bat" ด้วยสิทธิ์ Administrator
   3. ลองปิด Windows Firewall ชั่วคราวเพื่อทดสอบ

❓ ปัญหา: "Port already in use"
✅ แก้ไข: มีโปรแกรมอื่นใช้ Port 8000 อยู่
   - ปิดโปรแกรมที่ใช้ Port 8000
   - หรือแก้ไขใน app.py บรรทัด 1325:
     เปลี่ยน port = int(os.environ.get("PORT", "8000"))
     เป็น port = int(os.environ.get("PORT", "8080"))
